## 목적

트렐로의 기본적인 구조를 클론하며 React의 기본 개념을 학습하고 JSON 형태의 데이터 통신을 경험하는 토이 프로젝트입니다.

## 주요 기능

보드와 투두를 연결해 특정 투두가 연결된 보드에만 생성, 수정, 삭제할 수 있도록 함  
로그인 상태를 통한 화면 전환  
Json-server를 이용해 REST API 구현

## 사용기술

React  
React hooks  
Context Api  
Javascript  
CSS3  
HTML5  
json-server
